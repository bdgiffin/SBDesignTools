# Steel Bridge Design Tools
#### A collection of useful excel spreadsheets for automated design workflow management
---
###Includes:
- ####proto.BRIDGE
	- An automated CAD to SAP model conversion tool
- ####re.STRUCT
	- A member sizing optimization tool
- ####fore.SCORE
	- An automated design scoring spreadsheet

####Originally written for the following software versions:
- ####AutoCAD 2013
- ####SAP2000 v15
- ####Microsoft Excel 2007
